# Card import drop zone

Drop a folder here per set, e.g.  imports/wheel-of-time/
Put anything you have inside it:
  - images (any names/formats: jpg/png/webp)
  - your stats/notes in whatever form (a .txt, .md, .csv, spreadsheet, screenshot, even messy)

Then tell Claude the set name. Claude structures it into cards.json (asking about
anything unclear), imports it, and you massage the cards in the web editor.

Ideal-but-optional per card: name, mana cost, type line, power/toughness (or loyalty),
keywords, rules text, flavor, rarity (C/U/R/M/S/L), and which image is its art.
Missing fields get sensible defaults you can fix later.
